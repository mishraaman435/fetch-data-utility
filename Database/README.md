# MPSSDI Data Fetch Utility

A secure and read-only data retrieval system built to streamline database access for users without allowing data modification.

## 🚀 Tech Stack
- Frontend: Angular
- Backend: .NET Core
- Database: PostgreSQL

## 🔑 Features
- Secure data access using API
- Token-based authentication
- Query logging system
- Role & designation management
- Read-only database utility

## 📂 Project Structure
- `/database` → Database schema and setup
- `/backend` → .NET APIs
- `/frontend` → Angular UI

## ⚙️ Setup Instructions

### 1. Database Setup
- Go to `/database`
- Run `schema.sql`

### 2. Backend Setup
- Configure connection string
- Run .NET project

### 3. Frontend Setup
- Run Angular project using:
```bash
npm install
ng serve
